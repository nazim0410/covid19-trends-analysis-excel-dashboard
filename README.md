🦠 COVID-19 Global Dashboard
An Excel-based analytical dashboard tracking global COVID-19 trends from 2020 to 2023 — covering case volumes, test positivity rates, country-level death rates, and vaccination rollout using pivot tables and charts.

What it covers
SheetDescriptionPositiveTestQuarterly and monthly test positivity rates and new case volumes (2020–2023)CasesDeathRateTotal cases and death rates for the top 10 most affected countriesdataRaw daily records by country — cases, deaths, ICU patients, tests, positivity rate, vaccinations

Key findings

Global positivity rate peaked in Q1 2022 at ~20%, driven by the Omicron wave
US led total cases at 103M, followed by China (99M) and India (45M)
Brazil had the highest death rate among top-10 countries at ~1.87%
South Korea and Japan recorded the lowest death rates despite high case counts (~0.1% and ~0.22%)
Case volumes declined sharply from mid-2022 onward as testing reporting dropped globally


How to use

Download the .xlsx file
Open in Microsoft Excel or Google Sheets
Use the pivot table slicers to filter by year, quarter, or country
Charts update automatically with your selection


Data source
Based on the Our World in Data COVID-19 dataset — a publicly available dataset covering 200+ countries from January 2020 onward.

Tools used

Microsoft Excel — pivot tables, pivot charts, calculated fields
Data cleaning and aggregation performed within Excel
